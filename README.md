# NeRF Camera View

## How to use

```bash
pnpm install
pnpm run dev
```

## Supported Formats

- NeRF Blender Dataset (which includes the `transforms.json` file)

## Screenshots

![Screenshot 1](/doc/example.jpg)

## Features

- Display the camera frustum in 3D space.
- Display the camera view when hovering over the camera frustum.
